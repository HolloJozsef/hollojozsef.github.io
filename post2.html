Diploma thesys presentation

This project was started in order to keep track of progress on my diploma thesys.

Implemented the login and register using tokens saved in the sessionStorage. Also integrated the Google Maps API and made the website available in Romanian and English.

Users can leave comments, see profile details and change their profile picture.

Technologies:
  • - PHP
  • - Laravel
  • - Angular
  • - SQL
  • - HTML/CSS
